# ①課題名
場所日記

## ②課題内容（どんな作品か）
Firebase Realtime Database を使った投稿型Webアプリ。
訪れた場所を「場所名」「一言感想」「星5段階評価」で記録できる。
投稿後は一覧に表示され、場所名から自動生成した「Googleマップで見る」リンクで入力した場所名の検索結果が開くようになっている

## ⑤工夫した点・こだわった点
本当は写真の共有をしたかったが、storageの設定から難しかったため断念。
しかし、他に方法がないかgeminiと相談した結果、地図の共有ができることがわかったため、そちらに方針を変更した。
結果として、地図が開けた方が場所の記録としては適していると感じたため怪我の功名だった。
- 場所名を入力すると、Googleマップの検索リンクを自動生成するようにした。
- 星をクリックして5段階評価を選べるようにした。
- 投稿時に日時を保存し、新しい投稿が上に表示されるようにした。

## ⑥難しかった点・次回トライしたいこと（又は機能）
- Firebase Storage はプランの都合で使わず、今回は写真投稿ではなく地図共有機能に変更した。
- 次回はログイン機能を追加して、自分の投稿だけを編集・削除できるようにしたい。
- Google Maps API を使って、アプリ内に地図を直接表示する機能にも挑戦したい。
- 投稿する画面と一覧の画面を分けて表示できるようにしたい。
